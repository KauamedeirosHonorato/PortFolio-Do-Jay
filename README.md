# Portfólio – JayChouDev

Este é o portfólio pessoal criado para apresentar projetos, habilidades e informações de contato. O site foi desenvolvido com **HTML, CSS e JavaScript**, focando em um design limpo, responsivo e organizado.

## 🎨 Sobre o Projeto

O portfólio foi estruturado de forma modular, com cada sessão organizada em uma página ou arquivo dedicado. O objetivo é oferecer uma navegação clara e uma experiência visual agradável, com animações leves e interface intuitiva.

## 🚀 Funcionalidades

* Estrutura em múltiplas páginas
* Layout totalmente responsivo
* Navegação fluida com menu interativo
* Seções dedicadas: *Sobre mim*, *Projetos*, *Contato*, etc.
* Componentes reutilizáveis (menus, ícones, estilizações)

## 🧠 Tecnologias Utilizadas

* **HTML5** para estrutura
* **CSS3** para estilos, animações e responsividade
* **JavaScript Vanilla** para interações e manipulação de menu

1. Baixe ou clone o repositório:

Pronto! O site funciona apenas no front-end, sem necessidade de servidor.
